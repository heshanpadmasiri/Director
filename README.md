# Director
A simple keybord based file manager inspired by the [dired mode](https://www.gnu.org/software/emacs/manual/html_node/emacs/Dired.html) in Emacs.

## Keybindings

| Command                 | Key        |
|-------------------------|------------|
| Navigation up           | k          |
| Navigation down         | j          |
| Naviation back          | h          |
| Navigation forward      | l / return |
| Mark file               | m          |
| Open directory          | o          |
| copy files to directory | c           |


## Build

``` sh
npm run tauri build
```
